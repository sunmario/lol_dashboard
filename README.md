# [WIP]Дашборд по Ranked-матчам League of Legends
Аналитический дашборд по Ranked-матчам League of Legends для анализа эффективности игроков и популярности чемпионов.
Исследуемые метрики:
1. Игрок
- winrate
- KDA
- average deaths
- average assists
- average gold
- CS per minute

2. Чемпионы
- pick rate
- win rate
- ban rate
- средний KDA

3. Матчи
- длительность
- количество убийств
- snowball
- comeback

**Структура дашборда**
- Вкладка 1 - Общая статистика
    - количество матчей
    - средний winrate
    - средний KDA
- Вкладка 2 - Чемпионы
    - топ чемпионов
    - winrate
    - pickrate
- Вкладка 3 - Игроки
    - рейтинг
    - лучшие игроки
    - динамика MMR
